# ASA 7 Public Observatory Overview

## Purpose

This document explains the intended public observatory surface of ASA 7.

## Core Surface

The public observatory is built around three main views:

- `Live Signal`
- `Layer Hypothesis`
- `Experiment Matrix`

## Live Signal

The live signal surface is designed to show:

- active data file or probe
- current signal posture
- trajectory metrics
- fracture snapshot cards
- current interpretation in public-safe language

This surface is not meant to claim proof.
It is meant to show active signal reading.

## Layer Hypothesis

The layer hypothesis surface shows whether the current run looks:

- more locally stable
- more structurally constrained
- or still ambiguous

This is a research-facing interpretation surface, not a final truth engine.

## Experiment Matrix

The experiment matrix is the discipline layer.

It reminds the viewer that ASA 7 only becomes meaningful through:

- structural repetition
- pressure ramping
- perturbation tests
- replayable comparisons

## Reading Rule

ASA 7 should always be read through:

- reruns
- controls
- repeatability
- simpler explanations first

One evocative result is never enough.
