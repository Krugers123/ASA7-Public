# ASA 7 Public Protocol Overview

## Purpose

ASA 7 uses protocol logic to turn observer-layer behavior into a structured research surface.

The public layer does not expose full protocol internals.
It exposes only the main public-safe protocol logic.

## Public Protocol Families

### OSP

Observer Signature Protocol.

Used to identify whether recurrent observer-layer signatures remain visible across controlled runs.

### TTP

Threshold Transition Protocol.

Used to track when stable behavior begins to compress into repeatable threshold behavior.

### DGP

Directional Geometry Protocol.

Used to separate random drift from more patterned drift movement.

### RCP

Recovery Compression Protocol.

Used to read whether repeated perturbation reduces recovery flexibility.

### SCP

Structural Comparison Protocol.

Used to compare whether fracture patterns appear to depend more on structure than topic.

## Public Reading

Together, these protocol families allow ASA 7 to act as:

- an observer-layer observability surface
- a repeatability-focused structural research layer
- a disciplined comparison instrument for advanced AI boundary behavior
