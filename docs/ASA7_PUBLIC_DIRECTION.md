# ASA 7 Public Scope and Direction

## Purpose

This document explains the public-safe direction of ASA 7.

## Public Direction

ASA 7 is positioned as a frontier observability direction for studying:

- observer-layer boundary behavior
- recurrent structural signatures
- threshold transitions
- instability under pressure

The public direction is intentionally engineering-first.

## What ASA 7 Is Not

ASA 7 should not be framed publicly as:

- metaphysical proof
- simulation proof
- ontology theater
- a declaration about the nature of reality

The public project is strongest when it stays focused on:

- repeatable signals
- observer-layer behavior
- structured comparison
- boundary behavior in advanced AI systems

## Public Direction Rule

The project should sound:

- unusual
- technically disciplined
- falsifiable
- repeatability-oriented

It should not sound like an unbounded speculative claim.
