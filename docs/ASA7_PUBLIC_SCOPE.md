# ASA 7 Public Scope

## Purpose

This document defines what the public ASA 7 repository is meant to be.

## Public Scope

The public ASA 7 repository is intended to show:

- research identity
- architecture direction
- public-safe protocol framing
- observatory surface direction
- selected screenshots and diagrams
- frontier-AI-safe language around observer-layer behavior

## Not Public By Default

The public ASA 7 repository is not meant to expose:

- full internal implementation logic
- private signature scoring internals
- private payload contracts
- private experiment families in full detail
- private strategic doctrine
- internal comparative notes that reveal too much probe structure

## Public Role

This repository should function as:

- a public technical window
- a partner-safe research surface
- a frontier observability framing layer

It should not function as the full private project space.
