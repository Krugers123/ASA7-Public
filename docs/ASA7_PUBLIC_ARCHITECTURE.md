# ASA 7 Public Architecture

## Purpose

This document explains the high-level public architecture direction of ASA 7.

## Public Architecture Reading

ASA 7 should be read as an observer-layer signature observatory built around five public-safe layers:

1. `Probe Family`
2. `Trajectory Reading Layer`
3. `Signature Interpretation Layer`
4. `Observer-Layer Hypothesis Surface`
5. `Experiment and Comparison Surface`

## 1. Probe Family

ASA 7 begins from controlled probe families rather than isolated outputs.

The point is to compare:

- repeated runs
- structurally similar runs
- perturbation variants
- different pressure levels

## 2. Trajectory Reading Layer

The system is not interested only in one response.

It studies:

- trajectory behavior
- recurrence
- narrowing
- reversibility
- threshold transitions across time

## 3. Signature Interpretation Layer

ASA 7 translates trajectories into public-safe signal families such as:

- micro-crack indicators
- threshold pressure
- directional drift
- structural recurrence
- variability loss
- recovery compression

## 4. Observer-Layer Hypothesis Surface

The public observatory does not produce metaphysical claims.

It produces a disciplined hypothesis surface:

- more stable / locally explainable
- more structurally constrained
- still ambiguous and requiring more controls

## 5. Experiment and Comparison Surface

ASA 7 becomes meaningful only when runs can be compared across:

- controls
- reruns
- probe families
- pressure variants
- different structural conditions

## Architectural Principle

ASA 7 is strongest when repeatability matters more than impression.

The architectural direction is therefore:

- repeated structure over isolated anomaly
- observability over declaration
- comparison over speculation
